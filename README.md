# 💰 Compound Interest Calculator

A simple and clean compound interest calculator using HTML, CSS, and JavaScript.

## 🔢 Formula
A = P * (1 + r / 100) ^ t

## 📦 Files
- `index.html`
- `styles.css`
- `script.js`

## 🚀 Usage
Open `index.html` in any browser, enter values, and click **Calculate**.
